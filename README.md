# GestionPro

**URL del repositorio:** https://github.com/Organizacionseven7/GestionPro

App web para la administración y gestión de empresas o negocios.

## 🚀 Inicio Rápido

Abre `index.html` directamente en tu navegador — no requiere servidor ni instalación.

```bash
# Opción 1 – abrir directo
open index.html

# Opción 2 – servidor local con Python
python3 -m http.server 8080
# luego visita http://localhost:8080
```

## 📋 Módulos

| Módulo | Descripción |
|---|---|
| **Dashboard** | Resumen de clientes, productos, facturas e ingresos |
| **Clientes** | Alta, edición y eliminación de clientes |
| **Productos** | Gestión de catálogo con precio y stock |
| **Facturas** | Creación de facturas con múltiples ítems |
| **Reportes** | Ingresos del mes, top clientes y estadísticas |
| **Configuración** | Datos de la empresa (nombre, RUC, dirección, etc.) |
